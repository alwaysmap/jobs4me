**Jobs For Me** was built by [Dylan Thomas](https://bitsby.me) at [AlwaysMap](https://alwaysmap.com).

## Why this exists

Job searching is tedious. You check the same boards, read the same JDs, and try to remember which companies you've already looked at. It's the kind of repetitive, detail-heavy work that an AI agent can do well — if you teach it what you actually want.

Jobs For Me is a Claude plugin that runs your job search as a persistent agent. It learns your background, searches for roles, scores them against your profile, and builds a pipeline you can track. Every time you decline a role, it gets smarter. Every time you prep for an interview, it maps your specific experience to their questions.

## How it works

The plugin teaches Claude how to be a job search agent through **skills** (domain knowledge) and **commands** (actions). Your data lives as plain YAML and markdown files in a folder you control. No database, no server, no API keys — just Claude and your files.

## Open source

The source code is at [github.com/alwaysmap/jobs4me](https://github.com/alwaysmap/jobs4me). The plugin runs on your existing Claude Pro or Max subscription — no additional cost.
